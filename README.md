# SabTurno Client Web

Frontend web para clientes construido con React y React Router v7, reutilizando contratos y logica de la app Expo.

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Variables de entorno

```bash
VITE_API_URL=https://app-api.sabturno.com
```

## Alcance actual

- Login, registro y recuperacion de contrasena.
- Home cliente con proximo turno y metricas.
- Reserva completa: local, servicio, fecha, horario, metodo de pago y estado del pago.
- Mis turnos con cancelacion.
- Perfil, edicion de datos, pagos y eliminacion de cuenta.
